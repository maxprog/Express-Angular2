# Express + Angular2 with TypeScript 

## Installation

1. Install the dependencies in project root and /client directory 

```
npm install
```

2. Install bower dependencies in /client (boostrap and jquery lib)

```
bower install
```

3. compile .ts files in Angular. Open another command line and go to the /client folder and run

```
npm run tsc -w
```

4. Start the server from the root

```
node server
```

5. Visit http://localhost:3000
